# M1_Diego_Rey

El proyecto consiste en la plantilla de una tienda de venta de videojuegos de segunda mano. Las pantallas realizadas son:
    - Una pantalla que tiene la lista de productos y contiene botones que permiten realizar algunas acciones, como ver, editar o borrar un producto.
    - Una pantalla que permite ver un producto en detalle.
    - Una pantalla que permite editar los datos de un producto ya existente, así como crear uno nuevo.
    - Una pantalla adicional que sirve a modo de carrito de compra: permite elegir cuantos productos comprar y hace los cálculos. 
      No se puede exceder el stock ni poner stock negativo (la pagina se encarga de ello).
